# House of Ascend

Marketing website for **House of Ascend** - a luxury influencer marketing agency.
Tagline: *Ascend Without Limits*.

> We don't just market brands. We elevate them.

## About

A responsive, single-page marketing site built with plain HTML, CSS and JavaScript -
no build step and no dependencies, so it deploys to any static host as-is. The design
follows the brand's black-and-gold luxury identity.

## Structure

```
index.html        Page markup and copy
css/styles.css    Black & gold theme, layout, responsive rules
js/main.js        Sticky nav, mobile menu, scroll reveals, stat counters, contact form
assets/           Logo (PNG) and SVG monogram / favicon
```

## Run locally

```bash
python -m http.server 8000
# then open http://localhost:8000
```

Or simply open `index.html` in a browser.

## Deploy

Hosted with GitHub Pages from the `main` branch (root). Any static host
(Netlify, Vercel, Cloudflare Pages) works the same way.
